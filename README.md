# simpleQuiz

Um projeto de quiz interativo desenvolvido com **React**, **TypeScript**, **TailwindCSS** e **shadcn/ui**.  
O objetivo é oferecer uma experiência divertida e educativa com perguntas de múltipla escolha em diferentes categorias.

---

## 🚀 Tecnologias utilizadas
- [React](https://react.dev/) — Biblioteca para construção de interfaces.
- [TypeScript](https://www.typescriptlang.org/) — Tipagem estática para maior segurança no código.
- [TailwindCSS](https://tailwindcss.com/) — Estilização rápida e responsiva.
- [shadcn/ui](https://ui.shadcn.com/) — Componentes acessíveis e modernos.

---

## 📦 Instalação

Clone o repositório e instale as dependências:

```bash
git clone https://github.com/Jose-GuilhermeG/simpleQuiz/
cd simpleQuiz
npm install
```

## ▶️ Executando o projeto
Para rodar em ambiente de desenvolvimento:

```bash
cd app
npm run dev
O projeto estará disponível em:
http://localhost:5000
```

🛠️ Estrutura básica
src/components/ui → Componentes reutilizáveis (botões, cards, etc.)

src/questions → Base de dados das perguntas.

src/pages → Páginas principais do quiz.

src/styles → Configurações do TailwindCSS.

## 🎨 Estilo e UI
A interface combina TailwindCSS para estilização rápida com os componentes do shadcn/ui, garantindo:

- Design moderno
- Acessibilidade
- Responsividade
